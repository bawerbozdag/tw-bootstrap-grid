# Security Policy for tw-bootstrap-grid

## Supported Versions

We currently provide security updates for the following versions of `tw-bootstrap-grid`.

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅        |
| <1.0.0  | ❌        |

## Reporting a Vulnerability

Security is a priority for `tw-bootstrap-grid`.

If you discover a security vulnerability within this package, please **DO NOT** create a public issue on GitHub. Publicly disclosing a vulnerability can put other users at risk before a patch is available.

Instead, please report the issue privately by sending an email to [bawerbozdag@gmail.com](mailto:bawerbozdag@gmail.com).

**What to expect:**

- We will acknowledge receipt of your vulnerability report within 48 hours.
- We will investigate the issue and work on a secure patch.
- Once the vulnerability is resolved, a new patched version will be published.

Thank you for helping to keep `tw-bootstrap-grid` and its users safe!
